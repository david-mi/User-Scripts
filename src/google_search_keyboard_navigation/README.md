# Google Search - Keyboard Navigation v1.0.0

## Description

This script prevents simplify keyboard navigation upon google search results.

## Shortcuts

`CTRL` + `ARROW DOWN` : Focus next result link

`CTRL` + `ARROW UP` : Focus previous result link
